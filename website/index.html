Why Voxtype?

Built specifically for the modern Linux desktop

Works on All Wayland Compositors

Uses kernel-level input (evdev) instead of compositor-specific protocols. Works on GNOME, KDE, Sway, Hyprland, and any other Wayland compositor.

Fully Offline & Private

All speech recognition happens locally using whisper.cpp. Your voice never leaves your machine. No cloud, no subscriptions, no tracking.

Push-to-Talk

Natural workflow: hold a key to record, release to transcribe. No "wake words", no accidental activations, just controlled dictation.

Highly Configurable

Choose your hotkey, Whisper model size, output mode (type or clipboard), and more. Works the way you want it to.

Smart Fallbacks

Types directly at cursor via ydotool, automatically falls back to clipboard if typing isn't available. Always works, one way or another.

Fast & Lightweight

Written in Rust for performance. Single binary, minimal dependencies. Uses ~500MB RAM with the recommended model.

Installation

Get up and running in minutes

Install from AUR 
# Using paru
paru -S voxtype

# Or using yay
yay -S voxtype

# Install optional dependencies
sudo pacman -S ydotool wl-clipboard
Install .deb package 
# Download and install
wget https://github.com/peteonrails/voxtype/releases/download/v0.1.0/voxtype_0.1.0-1_amd64.deb
sudo dpkg -i voxtype_0.1.0-1_amd64.deb
sudo apt-get install -f

# Install optional dependencies
sudo apt install ydotool wl-clipboard
Install RPM package 
# Download and install
wget https://github.com/peteonrails/voxtype/releases/download/v0.1.0/voxtype-0.1.0-1.fc39.x86_64.rpm
sudo dnf install ./voxtype-0.1.0-1.fc39.x86_64.rpm

# Install optional dependencies
sudo dnf install ydotool wl-clipboard
Build from source 
# Install Rust (if needed)
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# Install build dependencies (Arch)
sudo pacman -S base-devel clang alsa-lib

# Clone and build
git clone https://github.com/peteonrails/voxtype
cd voxtype
cargo build --release

# Install
sudo cp target/release/voxtype /usr/local/bin/

Post-Installation Setup

# Add user to input group (required for hotkey detection)
sudo usermod -aG input $USER
# Log out and back in for group change to take effect

# Start ydotool daemon (for typing output)
systemctl --user enable --now ydotool

# Download whisper model and verify setup
voxtype setup --download

# Start voxtype!
voxtype
